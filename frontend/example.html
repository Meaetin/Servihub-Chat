ServHub Chat Widget Example

How to Embed the Chat Widget

To embed the ServHub chat widget in your website, follow these steps:

1. Include React and ReactDOM (if not already included)

<script crossorigin src="https://unpkg.com/react@18/umd/react.production.min.js"></script>
<script crossorigin src="https://unpkg.com/react-dom@18/umd/react-dom.production.min.js"></script>

2. Include the ServHub Chat Widget

<script src="./dist/servihub-chat-widget.js"></script>

3. Add a container element

<div id="my-chat-widget"></div>

4. Initialize the widget

<script>
  // Initialize the chat widget
  const chatWidget = ServiHubChat('my-chat-widget', {
    apiUrl: 'https://your-api-server.com',
    // Add other configuration options here
  });
</script>

Widget Features

  • ✅ Responsive message bubbles
  • ✅ Chat input with file upload support
  • ✅ Mobile-responsive design
  • ✅ Smooth animations and transitions
  • ✅ Typing indicators
  • ✅ Timestamp display
  • ✅ Embeddable as UMD bundle (≤20kB target)

Note: The chat widget will appear in the bottom-right corner of the page. Click the chat bubble to open it!
